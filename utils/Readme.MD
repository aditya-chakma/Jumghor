# Module: Utils

Module wraps all utils classes. Can be imported as dependency in other modules.

## Setup

| Module Name | Current Version |
|-------------|-----------------|
| Utils       | 1.0-SNAPSHOT    |

## Dependency

```xml
<dependency>
    <groupId>com.iAxis.jumghor</groupId>
    <artifactId>utils</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```
